# THOOS Terminal — Developer Documentation (Public)

**Official non-sensitive reference documentation for THOOS Terminal.**

This public repository contains only developer-facing reference materials for the **THOOS Terminal** (the live trading interface of the THOOS platform). It is intended for internal team collaboration, real-time review with Grok, and safe external developer/API discussions.

**Strict Scope – Terminal Only**  
No content related to THOOS Edge, Labs, Engine, or Admin will be included in this repository at this time.

**Permitted Content**
- Brand Guidelines
- UI Design System principles and specifications
- Chart configuration standards
- Notifications Engine reference (Terminal integration only)
- Engineering memos and workflow documentation (Terminal-specific)
- High-level, public-safe technical overviews

**Strict Prohibition**
- Never commit API keys, secrets, database schemas, RLS rules, full entity definitions, source code files, or any sensitive internal architecture.
- All files must be reviewed for compliance before pushing.
